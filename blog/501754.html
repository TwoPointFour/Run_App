The Retina display is the new �must have� feature of iPhone 4. While the display size still remains to be 3.5�, it has 4 times the number of pixels as previous models. This essentially means that what you see on screen is pixel perfect and crisper than ever before.
The 4k resolution means there are 4000 horizontal pixels in total, which is about 4 times the number of pixels as previous models. The pixel density of a 4k monitor is 218ppi, which is 88% more than a full HD (1920 x 1080) display.
A 4k monitor can be considered a good investment for all kinds of professionals. People who edit videos, for example, will definitely see a difference. For graphic designers, the higher screen resolution is a very welcome addition to their toolset. Programming is also improved with the ability to see more lines of code than before.
